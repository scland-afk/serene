## Keyword Strategy (Non-Spammy)
Rule: One page = one primary intent.

Per page usage guideline:
- Primary keyword: 1–2 mentions
- Secondary keyword or close variant: 1 mention
- Natural variation: 1 mention
- Total target mentions: 3–5 max

Placement order:
1. H1 (exact or close match)
2. First paragraph (natural)
3. One H2/H3 (variant or support)
4. One internal link anchor (clean, not stuffed)
5. Meta title/description (one clear mention)

Avoid:
- "near me" phrasing
- stacked keywords
- repeating the exact phrase in multiple consecutive lines

## Writing Prompt (Tone + Style)
Use this when drafting new content:

- Conversational, engaging tone with human flow
- Flesch Reading Ease around ~80
- Mix short and long sentences; avoid excessive adverbs
- Use mild repetition for emphasis; avoid mechanical patterns
- Avoid hype/marketing language
- Avoid these words: opt, dive, unlock, unleash, intricate, utilization, transformative, alignment, proactive, scalable, benchmark
- Avoid these phrases: "In this world," "in today’s world," "at the end of the day," "on the same page," "end-to-end," "in order to," "best practices," "dive into"
- Use simple language and be direct
- It's ok to start sentences with "and" or "but"
- Keep it real; don't force friendliness
